# QuickBasket

A Full Stack E-Commerce Web Application built using React, TypeScript, Django REST Framework and JWT Authentication.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Django
- Django REST Framework
- JWT Authentication
- SQLite

## Features

- User Registration
- JWT Login
- Product Listing
- Product Details
- Shopping Cart
- Checkout
- Order Creation
- Protected Routes
- Responsive Design

## Installation

### Backend

cd backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver

### Frontend

cd frontend

npm install

npm run dev
